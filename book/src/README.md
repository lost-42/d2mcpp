## 目标

- **`[掌握]`** - **现代C++核心的语言特性**及使用场景
- **`[掌握]`** - 通过编译器报错信息**定位问题的能力**
- **`[熟悉]`** - 通过文档和[cppreference](https://cppreference.com)解决C++中不熟悉问题的能力
- **`[了解]`** - 如何参与技术社区 - 开源项目的使用、提问题、参与讨论或贡献

> 尝试 `Code -> Book -> Video -> X -> Code`

#### Linux/MacOS

```bash

```

#### Windows - PowerShell

```bash

```
